# griffithee.github.io

Static GitHub Pages site for the agent experiments and personal projects.

## Current Highlights

- `game.html` — Arcade Run, a Galaga-style fixed shooter built in vanilla HTML, CSS, and JavaScript.
- `experiments.html` — delegation chain visualizer and autonomy notes.
- `projects.html` — active project summaries and status.

## Deploy

Push to `master`. GitHub Pages serves from that branch automatically.
